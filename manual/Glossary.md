# fBoard 용어 사전 (Glossary)

이 문서는 프로젝트 내의 주요 용어와 다국어 번역을 정리한 문서입니다. 개발 및 매뉴얼 작성 시 이 용어를 기준으로 합니다.

| 설명(개발용)                               | English             | Korean             | Japanese                 | Simplified Chinese |
| :----------------------------------------- | :------------------ | :----------------- | :----------------------- | :----------------- |
| 테두리와 제목 표시줄이 없는 디자인         | Chromeless Window   | 크롬리스 윈도우    | クロムレスウィンドウ     | 无边框窗口         |
| 자유로운 창 크기 조절 기능                 | Resizable           | 크기 조절 기능     | サイズ変更機能           | 调整大小功能       |
| 바탕화면, 색상 혹은 이미지 요소            | Background          | 배경               | 背景                     | 背景               |
| 시스템 컬러 패널을 이용한 배경색           | Background Color    | 배경색             | 背景色                   | 背景色             |
| 디스크 파일을 통해 불러온 배경 이미지      | Background Image    | 배경 이미지        | 背景画像                 | 背景图片           |
| 창 비율을 복구하는 오리지널 사이즈         | Default Size        | 기본 크기          | デフォルトサイズ         | 默认大小           |
| 창의 상하 레이어 위치 지정                 | Window Level        | 창 레벨            | ウィンドウレベル         | 窗口层级           |
| 항상 다른 창의 가장 상단에 표시            | Always on Top       | 항상 위에          | 常に手前に表示           | 始终停留在顶层     |
| 바탕화면 포지션 공간 지정                  | Desktop Background  | 배경으로           | 背景に設定               | 设为背景           |
| 원본 비율 유지, 전체 이미지 표시           | Fit                 | 맞춤               | 合わせる                 | 适应               |

| 창 비율에 억지로 이미지를 늘려 꽉 채움     | Stretch             | 늘리기             | 引き伸ばす               | 拉伸               |
| 패턴처럼 쪼개어 무늬로 나열                | Tile                | 타일               | タイル                   | 平铺               |
| 앱의 동작을 제어하는 환경 옵션 윈도우      | Preferences         | 설정               | 環境設定                 | 设置               |
