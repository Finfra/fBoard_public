> **나만의 맞춤형 스크린 보드로 모든 시선을 사로잡으세요.**

macOS 맞춤형 배경 도구. 아름다운 프레젠테이션 배경을 만들고, 데스크톱 프라이버시를 보호하며, 브랜드 컬러를 원클릭으로 적용하세요.

# 주요 기능

* **아름다운 배경** - 프레젠테이션을 위한 맞춤형 배경
* **프라이버시 보호** - 데스크톱 정보를 즉시 숨김
* **간편한 커스터마이징** - 단색, 그라데이션, 이미지 배경 지원
* **원클릭 정리** - 간단한 메뉴바 아이콘 활성화
* **컬러 매칭** - 브랜드 컬러에 맞춤 적용
* **멀티 디스플레이 지원** - 여러 모니터에서 동작
* **AI 에이전트 연동** - Claude, Gemini, MCP로 자동화

# AI 에이전트 연동

AI 에이전트로 fBoard를 자동화하고 확장하세요. 모든 연동 방식은 내장 REST API를 기반으로 합니다.

| 플랫폼     | 연동 방식                       | 상세                                     |
| ---------- | ------------------------------- | ---------------------------------------- |
| **Claude** | Marketplace Plugin (Skill 포함) | [Claude Code에서 설치](./agents/claude/) |
| **Gemini** | Workflow를 통한 설치            | [Gemini에서 설치](./agents/gemini/)      |
| **MCP**    | Model Context Protocol 서버     | [MCP 서버 설정](./mcp/)                 |

# 요구 사항

* macOS 12.0 이상
